> electron-demos 包含 electron 基本开发环境和一些常用功能极简示例

<br>

## 使用
下载安装
```bash
$ git clone https://github.com/haloislet/electron-demos.git
$ cd electron-demos
yarn
```

启动开发环境
```bash
$ npm start
```

构建生产环境
```bash
$ npm run build
```

发包到 github
```
$ npm run build -- -p always
```
